## Step 1: Create `.github/workflows/deploy.yml` — Done

**Files created:**
- `.github/workflows/deploy.yml` — GitHub Actions workflow triggered on push to `main`; uses `appleboy/ssh-action` to SSH into VPS as `deploy` user and run `git pull && docker compose up --build -d`

**Workflow details:**
- Triggers on push to `main` only
- Single job: `deploy`
- Uses `appleboy/ssh-action@v1` with host/key from repository secrets
- Runs as `deploy` user (not root)
- Commands: `cd /opt/apps/pidi && git pull && docker compose up --build -d`

---

## Step 2: VPS Setup — Not Started

**One-time setup required on VPS as root:**
1. Create `deploy` user (`adduser --disabled-password`)
2. Add to `docker` group
3. Generate dedicated SSH key pair (ed25519)
4. Set up `authorized_keys` for `deploy` user
5. `chown -R deploy:deploy /opt/apps/pidi`

See `docs/v2.2-plan.md` for exact commands.
