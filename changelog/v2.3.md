# V2.3 — Migrate to `create_agent`

Migrates both agents from `create_react_agent` (langgraph.prebuilt) to `create_agent` (langchain.agents), the recommended replacement. Same invoke API and result format, but adds middleware support and cleaner structured output for future use.

## What changed

- **Import swap** — `create_react_agent` → `create_agent` in root and web research agents.
- **Parameter rename** — `prompt=` → `system_prompt=` in both agent constructors.
- **Dependencies upgraded to 1.0 LTS** — added `langchain>=1.0,<2` and `langchain-core>=1.0,<2`; bumped `langgraph` to `>=1.0,<2`; pinned `langchain-community` to `<0.5`.

## Notes

- No changes to `get_response()`, Slack handlers, system prompts, tools, or Docker setup.
- `langgraph` kept as explicit dependency — used transitively by `create_agent`.
