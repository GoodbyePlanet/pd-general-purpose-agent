# V1 — Initial Release

Internal Slack AI agent for PD company. Responds to @mentions and emoji reactions using GPT-4o.

## What's included

- **Config** — pydantic-settings loading from `.env`; fast-fail on missing secrets
- **FastAPI** — single `/health` endpoint; uvicorn with `workers=1` (required for Socket Mode)
- **Slack integration** — Socket Mode (no public URL needed); handles three triggers:
  - `@mention` — replies in thread with full conversation history
  - Emoji reaction — replies in thread to the reacted message
  - Thread message — bot continues conversation in any thread it's already participating in
- **LangGraph agent** — `create_react_agent` with GPT-4o; empty tools list (ready for V2 additions)
- **Docker** — `python:3.12-slim` + `uv`; `restart: unless-stopped`; healthcheck on `/health`
