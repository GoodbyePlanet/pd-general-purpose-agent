# V2.4 — Switch from OpenAI to Anthropic

Replaces GPT-4o (OpenAI) with Claude (Anthropic) as the LLM provider for both the root agent and the web research sub-agent.

## What changed

- **LLM provider swap** — `ChatOpenAI` → `ChatAnthropic` in both root and web research agents.
- **Config renamed** — `openai_api_key` / `openai_model` → `anthropic_api_key` / `anthropic_model` in `config.py`.
- **Default model** — `gpt-4o` → `claude-haiku-4-5-20251001`.
- **Dependency swap** — `langchain-openai` replaced with `langchain-anthropic` in `pyproject.toml`.
- **Environment variables** — `OPENAI_API_KEY` / `OPENAI_MODEL` → `ANTHROPIC_API_KEY` / `ANTHROPIC_MODEL`.

## Thinking/Loading Indicator

Added a loading message that appears immediately when the bot starts processing. A random "thinking" message is posted in the thread, then updated in-place with the actual response via `chat_update`.

- **Random thinking messages** — 5 variations with different emojis (⏳ 🧠 🤔 🔍 ⚙️) picked at random.
- **Applied to all handlers** — `app_mention`, thread message, and `reaction_added`.
- **Error handling** — if the agent fails, the loading message is updated with the error text instead of posting a second message.

## Notes

- No changes to agent logic, tools, system prompts, or Docker setup.
- `.env.example` already had Anthropic variables — now the code matches.
